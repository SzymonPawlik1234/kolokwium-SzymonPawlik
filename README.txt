Aby uruchomić ten projekt, wykonaj poniższe kroki:

Upewnij się, że masz zainstalowane środowisko Node.js
 na swoim komputerze. Możesz pobrać je ze strony oficjalnej
 witryny Node.js: 
Sklonuj lub pobierz projekt z repozytorium,
 jeśli jeszcze tego nie zrobiłeś.

Otwórz terminal lub wiersz polecenia 
i przejdź do katalogu projektu.

Zainstaluj zależności, wykonując polecenie 
npm install. Spowoduje to pobranie wszystkich
 potrzebnych pakietów i bibliotek wymaganych do uruchomienia projektu.

Po zakończeniu instalacji, wykonaj polecenie
npm run dev, aby uruchomić serwer deweloperski. 
Proces ten wykorzystuje narzędzie Vite, które jest odpowiedzialne 
za kompilację i uruchamianie projektu.
